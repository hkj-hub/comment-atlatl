# BaseWrapper

- 指定した Width,Height を十字に仕切る線を BackgroundColor のグラデーションを使って表現
- 子要素を Width,Height の中央に配置
  - 物理演算の射影の枠として利用。物理演算では、各要素は(0,0)の原点からの位置を計算する。
  - 高さ幅 0 で position:relative、overflow:visible の原点要素を配置し、原点要素の中に position:absolute の物理演算射影要素を配置する。
  - 物理演算射影要素は transform で位置を指定する。
